# D133_Phone
基于D133的智能手机，非安卓

## :ledger: Overview

**Intro**<br>
D133-based smartphone, not Android.

**Argument**<br>
400MHz CPU,1M SRAM,8M psram.

**Screen**<br>
480*854 pixel, mipi port.

## :rocket: Details

**D133CBS**
  - sdcard.
  - touch.
  - audio.
  - IR led.
